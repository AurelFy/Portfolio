Working in a group of four, we were asked to create an interactive user experience in the form of a responsive website, using a dataset that we either found or created ourselves.

— Music meets clash... Where sound ignites tension and sparks curiosity.

02

02 02

We chose to let users compare two artists from the same musical genre across several purely statistical categories. We decided to focus on Rock and US Rap, selecting six artists for each. We created our own dataset for this purpose.

— We opted for a professional design — such as the music itself : clear, powerful, and unapologetic.

03

03 03

The design of this site is clean and minimalist to give it a professional feel. We used white on a black background. To clearly show the user the winner in each category when interacting with the data, we decided to highlight their scores in red.

— JavaScript, JavaScript and JavaScript again. This is the best way to learn it !

04

04 04

Before coding, we had to create a JSON file containing our data. Then, we used JavaScript to fetch and display this data on the web page. This process helped us deepen our knowledge and discover new skills. Since we were working as a group, using GitHub was essential.
